🔐 Passman

Secure Local Password Manager in Python

AES-256-GCM Argon2id KDF Zero Plaintext CLI-Based

Overview

Passman is a secure, local password manager built in Python with military-grade encryption. No plaintext ever touches disk, and your master password never leaves your machine. Complete offline operation ensures your secrets stay private.

🔒 Military-Grade Encryption

AES-256-GCM encryption with authenticated encryption ensures confidentiality and integrity.

🔑 Advanced Key Derivation

Argon2id with 600,000 iterations (OWASP 2023+ recommendation) protects against brute-force attacks.

💾 Atomic Operations

Atomic file writes with 0600 permissions prevent data corruption and unauthorized access.

🎲 CSPRNG Generation

Cryptographically secure password generation using Python's secrets module.

📋 Smart Clipboard

Auto-clearing clipboard with 25-second countdown to prevent accidental exposure.

🛡️ Zero Plaintext

No sensitive data ever written to disk in plaintext form.

Tech Stack

Python 3.8+ cryptography argon2-cffi pyperclip

Installation

git clone https://github.com/grisheet/passman.git cd passman pip install -r requirements.txt python -m passman.cli --help

Quick Start

# Initialize vault python -m passman.cli init # Add new password python -m passman.cli add # Retrieve password (copies to clipboard) python -m passman.cli get # List all entries python -m passman.cli list # Generate secure password python -m passman.cli generate --length 32

Security Features

  • ✅ No Network Access: Completely offline operation
  • ✅ Memory-Safe: Secure handling of secrets in memory
  • ✅ AAD Protection: Vault header bound to ciphertext prevents downgrade attacks
  • ✅ Fresh Nonces: New nonce generated for every encryption operation
  • ✅ Secure Defaults: No weak configuration options available
